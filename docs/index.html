About This Training

Background, what to expect, and what you will be able to do after.

The 1k1RG Database Curation and Loading Workshop is a two-day, hands-on technical training for IT personnel responsible for operating and maintaining the SNP-Seek rice genomics platform. SNP-Seek runs on a specialized stack — PostgreSQL with the Chado biological schema, HDF5 for genotype matrices, and brAPI for external system integration. This training closes the gap between server administration and the domain-specific knowledge required to independently curate, load, and verify genomic data without constant support from the development team.

Day 1

Database Schema, Ontologies & Curation

  • Platform overview and SNP-Seek technology stack
  • Chado schema — core tables, relationships, and ERD walkthrough
  • Biological ontologies and how cvterms act as the database type system
  • Hands-on: restore and query a real training database
  • Bulk data loading via script — step-by-step lab
  • HDF5 introduction — what genotype matrix files are and why they exist
Day 2

Pipeline, Loading & Verification

  • VCF format — what it is and what the genotype calls represent
  • Full two-step pipeline: VCF → genotype matrix → HDF5
  • Loading into Chado using uploadDataset.sh
  • Diagnosing and recovering from partial load failures
  • brAPI — verifying that loaded data is visible to external systems
  • Open lab: end-to-end run on an unseen dataset

Learning Outcomes

  • Navigate and query the SNP-Seek Chado database
  • Understand how ontologies and cvterms structure genomic data
  • Run the full VCF → HDF5 → Chado loading pipeline
  • Diagnose and recover from partial load and pipeline failures
  • Verify loaded data through brAPI endpoints
  • Perform routine maintenance: backup, restore, VACUUM, monitoring

Prerequisites

  • Linux command-line comfort — navigation, file operations, running scripts
  • SQL fundamentals — SELECT, JOIN, INSERT, basic relational tables
  • SSH access to the training server (credentials distributed before the session)
  • A laptop with an SSH client and a browser
  • No prior bioinformatics background required — biological concepts are introduced as they become relevant
