Session Overview

A detailed walkthrough of the Chado relational schema: its module structure, how it models biological entities (organisms, features, stocks, phenotypes), and specifically how it encodes ontology terms and their relationships. Directly prepares participants for the Restore & Explore lab that follows.

Topics Covered (Placeholder)

  • Chado schema overview and module map (organism, sequence, stock, phenotype, cv, pub)
  • The cv and cvterm tables — storing ontology terms
  • The feature table and featureprop
  • The stock and stockprop tables for accession data
  • Cross-references via dbxref
  • How SNP data maps to Chado entities
  • Common gotchas and SNP-Seek customisations
